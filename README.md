## 📱 Body Health Calculator
A simple Flutter application for calculating BMI and BMR.

## 🚀 Features
- Calculate BMI
- Calculate BMR
- Clean mobile UI
- Bottom navigation (BMI / About / BMR)
- Splash screen

## 🛠️ Tech Stack
- Flutter
- Dart

## 🧮 Formula
BMI
 - weight (kg) / height (m)^2

BMR
 - Male: 66 + (13.7 × w) + (5 × h) - (6.8 × age)
 - Female: 665 + (9.6 × w) + (1.8 × h) - (4.7 × age)
